FeedyUI

FeedyUI is an open-source, light-weight library of UI components based on the vanilla CSS. It includes a vast varaities of classes to support common html components used in day to day development.

Installation

FeedyUI has a few easy ways to quickly get started, each one appealing to a different skill level and use case.


The awesome folks over at Netlify have my back at provide CDN support for FeedyUI's CSS and JavaScript. Just use this FeedyUI CDN links and start your development journey.
